## 高爾夫物理學的簡易模型
## The Simple Golf Physics Model for Kids

我們建立一個使用人工智慧輔助高爾夫學習的簡易物理模型。
(化繁為簡的數學觀念 #小波正交 #簡易傅立葉轉換)

我們建立一個適合國小學生的高爾夫簡易物理模型，目的是用最少的物理概念解釋球為什麼「飛得遠、飛得高、飛得直」。模型以三個核心變數為基礎：速度、角度與方向。第一，球飛行距離主要與起飛速度有關，速度越大，距離幾乎呈平方增加；第二，起飛角度影響飛行高度與最佳距離，角度過低或過高都會降低飛行效果；第三，方向偏差來自桿面與揮桿路徑誤差，若產生側向速度，飛行時間越長偏移越大。我們將整個揮桿動作分解為六個階段，利用人工智慧擷取身體姿勢，計算如肩髖旋轉角度、重心轉移與頭部穩定等指標，將其對應到速度、角度與方向三個物理量。透過量化指標與實際落點比較，驗證模型的預測能力，並作為學習改進依據。
