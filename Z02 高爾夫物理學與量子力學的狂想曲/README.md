#量子力學：波函數在位置空間（座標）與動量空間（頻率）之間的切換，完全依賴傅立葉轉換。您可以透過 PhET 互動模擬 (Wave Packet) 來觀察多個正弦波是如何「堆疊」出一個物理粒子的位置。

#空間座標與能量頻率的轉換 #從揮桿的空間座標直接換算成高爾夫球的能量 #錯誤的揮桿就是把大部份能量都轉換成雜訊

在 #海森堡理論 中，系統的哈密頓量是位置和動量的函數，但它們不再具有 #古典力學 中的定義，而是由二階（代表著過程的初態和終態）#傅立葉 係數的 #矩陣 給出。海森堡還發現，這些矩陣互不對易。這些論述後來發展成為 #矩陣力學。

實際而言，許多宏觀系統都是用古典理論（如 #古典力學 和電磁學）來做精確描述。因此在非常「大」的系統中，#量子力學的特性應該會逐漸與古典物理的特性相近似，兩者必須相互符合。
